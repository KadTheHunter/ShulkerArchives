Fabric Mods

These mods are extremely useful when working with NBT and Mapart.

Armor Statues

Armor Statues is an excellent way to edit every detail about an armorstand while in-game, and all using a nice, clean GUI.

FZMM

FZMM is a somewhat niche but still incredible mod, offering tons of advanced features like creating Lore Art/Hologram Art, Modifying head Data to add custom textures, a GUI for designing Banners, and an excellent array of tools to give text everything from simple colors to beautiful gradients.

Important Note: As of ~1st August 2023, it seems the author and sole developer of FZMM has gone AWOL; Nobody is 100% sure what happened to him, only that he's seemingly vanished. The chance of FZMM receiving updates past 1.20.1 is uncertain.

Hotbars+

Hotbars+ allows you to have an infinite number of hotbars, by creating new hotbar files, and swapping them out at the click of a button. It may not be the easiest for on-the-go item storage (for that, see NBT Editor) but it's perfect for making sure your items and kits are safely stored, and accessible from anywhere in your Minecraft instance.

IBE Editor

IBE Editor (Item Block Entity Editor) is a Mod with similar features to NBT Editor (NBT/SNBT editing, Item Vault, etc.), though it really shines with its NBT Tag Editing page. No other mod in modern times displays NBT Tags in such a nice, easy to read Tree format besides IBE Editor, and that alone makes it worthy of including in your Minecraft instance.

NBT Editor

NBT Editor is the pinnacle of NBT Editing Mods. What it lacks in UI design, it makes up for a thousand times over in other features, such as the ability to Import and Export .nbt files, retrieval of "lost" items via its /get lostitem history command, and its amazing Client Chest, which by default has 100 large chests' space for items, but can be expanded infinitely. All of its features also fully support tags such as Infinity and NaN, which means you can safely use it without worrying about your tags being messed with.

NBT Void

NBT Void logs every item with custom NBT that your client encounters. Whether you want a form of backup in case you accidentally lose something, or you want a rudimentary ItemLogger, NBT Void can do it-- just as long as you don't want to save Infinity/NaN tags, because it doesn't currently support that :(

Symbol Chat

Symbol Chat adds a menu to Chat, Signs, Books, and most other areas of Text Input that allows you to use different fonts, and all manner of unicode symbols.

WNT

WNT is the ultimate combination of Utility and Protection in Minecraft. Using its commands, or the built-in Blackbox menu, you can dump almost any data you choose, be it Entities, Maps, Threads, etc., and if you don't want all of it, you can choose a single specific Entity or Map to dump. In addition to this (and perhaps more importantly), it protects against the worst NBT Exploits, and allows you to still have some control over your game even when Minecraft has frozen.

Websites

These websites are excellent tools or guides, though many have better forms in Mods

Chat Symbols

Chat Symbols has a list of symbols, emoticons and characters supported by Minecraft for Books, Signs and Chat.
Alternatively: Symbol Chat

Colorize.fun

Colorize.fun lets you create colored text, and provides versions for Minecraft as well as HTML, BBCode, and IRC.
Alternatively: FZMM

Mapart Craft

MapartCraft is the do-it-all site for Mapart, where you can upload an image and customize the dithering, size, blocks used and other options, before either saving the preview image or downloading the map as a Schematic (.nbt) or Data (.dat) file.

MCStacker

MCStacker is an online Command/NBT generator that allows you to choose the command and features, and then generates it for you.
Alternatively: FZMM, IBE Editor, NBT Editor

Minecraft-ArmorStand

Minecraft-ArmorStand is an excellent way to pose armor stands and test out armor and tags before generating a command for them.

NBT Iceberg

NBT Iceberg is a chart recording knowledge of NBT displayed by how common it is, with things like Chunk Bans and 127 stacked items being near the top as "well known", and things like "The Paper" and "2B Health" at the bottom as "completely obscure"

NBT Minifier

NBT Minifier is an experimental compressor to remove un-needed/blank/duplicate tags from SNBT, such as ones added by ItemizerX

Obj To Schematic

Obj To Schematic converts 3D model files such as .obj and .glb files into .schematic, .schem, .litematic, .nbt and .json files, with tons of settings to tweak and customize the output.
There is also an App version of this site available at LucasDower/ObjToSchematic on GitHub.

R74n

R74n is an interesting amalgamation of resources and tools for Minecraft and other things. Of particular note is the hidden Box Page containing the /give commands and NBT for a number of Shulker Kits, including Ryan UwU 1-7.

webNBT

webNBT lets you upload a .nbt file and edit it just as you would with an External Editor such as NBT Studio.

Applications & External Tools

These applications are typically only used when Mods don't exist or don't work.

Epsilon Bot

Epsilon Bot turns your chosen Minecraft account into a mapart building bot, with the bonus function of being able to sync with a local build server.
Check out our guide on running Epsilon Bot!

map-to-img

map-to-img is a little known tool for converting Minecraft maps to images.

NBT-Studio

NBT Studio is one of the best External NBT Editors, capable of editing any .nbt file with ease. Knowledge of NBT Tags will be needed to use it effectively.
